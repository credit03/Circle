# Circle
##使用Rxjava2+retrofit2+mvp仿写的一个朋友圈demo，使用go+beego开发的服务器
###仿微信(weixin wechat)实现的朋友圈，后台交互实现了点赞、评论、删除等功能，RecyclerView可以根据键盘的显示或隐藏实现联动。增加短视频拍摄功能，与后台交互采用mvp模式。关键字：wechat weixin 微信朋友圈
